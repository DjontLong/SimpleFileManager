```
**SimpleFileManager** - это простой файловый менеджер на языке Java, который позволяет выполнять основные операции с файлами и директориями через консольный интерфейс. 
Программа поддерживает создание, удаление, переименование файлов и папок, а также просмотр их содержимого.

Возможности:
- Создание папок:
В текущей директории или по указанному пути

- Создание файлов:
В текущей директории или по указанному пути

- Удаление файлов и папок:
Возможность удаления как файлов, так и непустых директорий

- Переименование файлов и папок:
- Указание нового имени для существующего файла или папки

- Просмотр содержимого директорий:
Отображение списка файлов и папок в указанной директории с информацией о размере и дате последнего изменения


```
